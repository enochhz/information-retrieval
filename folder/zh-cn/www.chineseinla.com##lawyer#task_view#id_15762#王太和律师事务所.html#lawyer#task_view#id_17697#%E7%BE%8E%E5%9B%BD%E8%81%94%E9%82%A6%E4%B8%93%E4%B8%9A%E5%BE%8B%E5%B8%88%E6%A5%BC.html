    王太和先生简历

    王太和,男,1964年7月20日出生于湖南省衡阳市城南区(现雁峰区), 现为美籍华人，系美国执业律师，是美国王太和联合律师事务所主任律师丶美国南加州湖南同乡联谊会法律顾问，烟台大学法学院兼职教授和中国上海政法学院客座研究员。

    1981年至1985年在武汉大学英文系学习，获文学学士学位；1985年至1987年在北京大学研究生院法律系国际法专业学习；1991年至1993年在美国威斯康辛大学法学院学习，获法学硕士学位。1993年至1997年在美国史考特律师事务所工作，1997年至现在，自己创办并经营的“王太和联合律师事务所”，任主任律师。该所现有员工15人，其中有3名美籍联合办案律师，4名翻译，和7名律师助理。 这在美国的律师事务所行业中， 规模为中型。

    到美国以后，多次积极参加为祖国抗险救灾活动及为中小学生义务教育募捐； 义务为国内有关高校讲学， 培养研究生及联系美国有关高校的学术交流与合作。 另外，还协助国内数千个家庭成功移民美国及在美安居立业;解决了相当一批中国企业和公司在美国生产和经营的各种法律问题。

    王律师在美国南加州各界有着广泛的人脉，良好的口碑和很高的社会声誉。
